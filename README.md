# The Zig installer on Windows

Documentation in two languages / Документация на двух языках

## English

The repo contains the Zig language installer version 0.16.0

The installer was made on the basis of Inno, and is focused on Windows x86-64

The Zig `0.16.0` folder contains the source files of the installer in Inno 7

Guide on how to install the installer:
1. You must have 7zip installed, or something like that
2. Download 3 files named `Zig Installer.7z.*`
3. Unzip the `Zig Installer.exe `from the archive of `Zig Installer.7z.001`
4. Run the `Zig Installer.exe `

You need to install all 3 files, but open and unzip only 1 - `Zig Installer.7z.001`


## Русский

В репо представлен установк языка Zig версии 0.16.0

Установщик был сделан на базе Inno, и ориентирован на Windows x86-64

В папке Zig `0.16.0` находятся исходные файлы установщика в Inno 7

Гайд, как установить установщик:
1. У вас должен быть установлен 7zip, или что то вроде того
2. Скачайте 3 файла с названием `Zig Installer.7z.*`
3. Разархевируйте `Zig Installer.exe` из архива `Zig Installer.7z.001`
4. Запускайте `Zig Installer.exe`

Установить нужно все 3 файла, но открыть и разархивировать только 1 - `Zig Installer.7z.001`
